# 🏠 Airbnb Clone (MERN Stack Project)

## 📌 Project Overview
This is my first **MERN Stack project**, a basic Airbnb-like web application where users can perform full **CRUD operations** on property listings.

The project demonstrates how a real-world booking platform works at a basic level, including creating, reading, updating, and deleting property data.
